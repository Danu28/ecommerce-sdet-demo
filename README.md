# 🧪 E-Commerce SDET Demo Project

## 🚀 Overview
Full-stack demo application built for learning test automation.

## 🏗️ Tech Stack
- Backend: Spring Boot
- Frontend: HTML, CSS, JS
- Database: H2
- Automation:
    - Selenium (UI)
    - Rest Assured (API)
    - TestNG

## 🔥 Features
- User Authentication (Register/Login)
- Product Management
- Cart System
- API + UI Integration

## 🧪 Automation Coverage
- UI Tests (Selenium)
- API Tests (Rest Assured)
- Integration Testing

## ▶️ How to Run

### Backend
```bash
cd backend
mvn spring-boot:run
